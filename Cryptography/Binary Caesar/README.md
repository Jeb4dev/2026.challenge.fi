# Binary Caesar

## Challenge

One of the oldest forms of encryption is the Caesar cipher, where each letter is shifted using a secret key.
This time, someone has modified the Caesar cipher by using XOR instead of addition.

Can you decrypt the flag?

Remember to insert the result in the flag format.
(ddc{example_flag} -> DDC{example_flag})
